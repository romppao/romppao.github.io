# VRTT Mario - Portfolio de Filmmaker y Fotógrafo

## Descripción

Este es un portfolio profesional replicado de https://vrttportfolio.com para Mario Espinoll Gonjar, un filmmaker y fotógrafo basado en Madrid, España.

## Características del Sitio

### Páginas Incluidas
- **index.html** - Página principal con secciones de hero, videos, fotos, sobre mí y contacto
- **videos.html** - Página dedicada a mostrar el portfolio de videos organizado por categorías
- **fotos.html** - Galería de fotos con filtros por categoría
- **contacto.html** - Formulario de contacto completo con información de contacto

### Características Técnicas
- **Diseño Responsivo** - Optimizado para todos los dispositivos
- **Animaciones Suaves** - Efectos de fade-in y transiciones
- **Modal Interactivos** - Para videos y fotos
- **Filtros de Categoría** - En la sección de fotos
- **Formulario de Contacto** - Completo con validación
- **Navegación Suave** - Scroll animado entre secciones

### Estilo Visual
- **Paleta de Colores** - Tonos neutros y elegantes con acentos dorados
- **Tipografía** - Inter font family para un look moderno
- **Efectos Visuales** - Gradientes, sombras y animaciones
- **Diseño Limpio** - Minimalista y profesional

## Instalación y Uso

### Para GitHub Pages

1. **Crear repositorio en GitHub**
   - Crea un nuevo repositorio con el nombre `tuusuario.github.io`

2. **Subir archivos**
   - Sube todos los archivos HTML a la rama principal
   - Asegúrate de que `index.html` esté en la raíz

3. **Activar GitHub Pages**
   - Ve a Settings > Pages
   - Selecciona la rama principal como fuente
   - Tu sitio estará disponible en `https://tuusuario.github.io`

### Personalización

#### Cambiar Información Personal
- Edita los archivos HTML para cambiar:
  - Nombre y título
  - Información de contacto
  - Descripciones de servicios
  - Textos de secciones

#### Agregar Contenido Multimedia

**Para Videos:**
- Reemplaza los thumbnails en las cards de video
- Actualiza los atributos `data-video` con los IDs de tus videos
- Modifica el reproductor de video modal según tus necesidades

**Para Fotos:**
- Reemplaza los gradientes de las cards de foto con imágenes reales
- Actualiza la información en los atributos `data-photo`
- Modifica el modal de fotos para mostrar tus imágenes

#### Estilos Personalizados
- Los estilos están integrados en cada archivo HTML
- Busca y modifica las variables de color en el CSS
- Ajusta animaciones y transiciones según preferencia

### Estructura de Archivos

```
/

├── fotos.html          # Página de fotos
```

- Las imágenes y videos son representaciones con gradientes
- Para producción, reemplaza con contenido multimedia real
- El formulario de contacto muestra una alerta (no envía emails realmente)
